1. Define Strings

String is a Collection of Characters Enclosed with a Single Qoutes or Double Qoutes

2. String Length

    EX :  let myStr = "Rakshitha M";
    let len = myStr.length;
    console.log(len); 
                            // o/p : 9

3. Concatination

It concat two string.

     var t1="Rakshitha";
     var t2="M";
     var t3=t1.concat(" ",t2);
     console.log(t3);         //o/p: Rakshitha M

4. Trim

It removes unnecessary spaces.

    let str="           Rakshitha";
    console.log(str.len);
    let tr = str.trim(); //20
    console.log(tr);        //Rakshitha
    console.log(tr.len);         //9

5. Reverse And Join

It reverse the string & It converts array to string.

    let rev1 = myStr.split("").reverse();
    console.log(rev1);  // ['e','x','t','r','a']
    let rev2 = myStr.split("").reverse().join("");
    console.log(rev2);              // o/p: artxe
